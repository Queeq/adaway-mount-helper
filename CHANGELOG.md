# AdAway Mount Helper
Minimal mount helper for AdAway compatibility with APatch, KernelSU and Magisk.

---

# Changelog

### 1.0.0
- Initial release: minimal mount helper extracted from bindhosts
- Stripped down to essential mount functionality only
- Removed WebUI, ad-blocking engine, automatic updates, and cron scheduling
- Kept operating mode detection for various root managers
- Support for SUSFS, ZygiskNext, and denylist handlers
- AdAway compatibility layer maintained
- Automatic release workflow enabled
